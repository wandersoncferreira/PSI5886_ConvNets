O que seriam variveis categoricas? Digamos que temos apenas 3 classes de nmeros em um determinado problema. "REAL, INTEIRO, IMAGINRIO".

A lista de targets poderia ser: [1.2, 2, 3, 4.2, 4i] No lugar de passar a lista acima, poderiamos categorizar em uma matriz onde cada linha mostra uma representao binarizada da varivel target.

target REAL INTEIRO IMAGINARIO
1.2 1 0 0
2 0 1 0
3 0 1 0
4.2 1 0 0
4i 0 0 1

Assim, passamos essa matriz binarizada para a rede. Vrias bibliotecas tem implementada essa funcionalidade. O pandas usa o mtodo .get_dummies(), no Keras podemos usar um mtodo chamado to_categorical

Finalmente, podemos criar o modelo de rede neural para o problema de reconhecimento de digitos. No Keras tem uma gama bem grande de modelos prontos, basta instanciar e montar as camadas.

O modelo de rede neural convolucional apresenta bsicamente os seguintes componentes:

  1. Uma camada convolucional
  2. Uma camada de Pooling
  3. Uma camada completamente conectada
  4. Uma camada completamente conectada com Softmax como funo de ativao

Acho que nessa estrutura podemos avanar mais sobre o entendimento delas. Deixar mais claro o que cada uma est fazendo, com exemplos visuais se possvel. E um pouco de intuio sobre o fenmeno.

class LossHistory(keras.callbacks.Callback):
    def on_train_begin(self, logs={}):
        self.losses = []

    def on_batch_end(self, batch, logs={}):
        self.losses.append(logs.get('loss'))

#
